# AI-Based Recipe Generator using Generative AI

## Problem Statement
To design and implement a Generative AI system that generates cooking recipes based on user-provided ingredients.

## Description
This project uses a pre-trained Generative AI model to create structured recipes including dish name, ingredients, and step-by-step cooking instructions.

## Technologies Used
- Python
- Hugging Face Transformers
- GPT-2

## How to Run
1. Open the notebook
2. Run all cells
3. Enter ingredients to generate a recipe

## Output
Generates a complete recipe using Generative AI.
